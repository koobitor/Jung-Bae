Jung-Bae
========

Mapping Jung Bae

How To Use 
- git clone https://github.com/koobitor/Jung-Bae.git
- cd Jung-Bae
- npm install
- node-dev server.js

Version 0.0.1
- Mapping Array
- '/wtr/:word' Change Jung-Bae to Thai
- '/rtw/:word' Change Thai to Jung-Bae
